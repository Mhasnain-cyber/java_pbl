StringBuffer Operations
append() ▼

Appends text to the end of the StringBuffer.

insert() ▼

Inserts text at the specified index.

delete() ▼

Deletes characters from start (inclusive) to end (exclusive).

reverse() ▼

Reverses the entire StringBuffer character sequence.

replace() ▼

Replaces characters from start to end with new text.

length() ▼

Returns the number of characters in the StringBuffer.

charAt() ▼

Returns the character at the specified index.

StringTokenizer Operations
tokenize() ▼

Splits the string using the delimiter. Iterates with hasMoreTokens() + nextToken().

countTokens() ▼

Counts tokens without consuming them from the tokenizer.

💡 Key Concepts

StringBuffer

  • append(str) — Add to end
  • insert(i, str) — Insert at index
  • delete(s, e) — Remove range
  • reverse() — Flip characters
  • replace(s, e, str) — Overwrite range
  • charAt(i) — Get char at index
  • length() — Size of buffer
✅ Mutable & Thread-Safe

StringTokenizer

  • new StringTokenizer(str, delim)
  • hasMoreTokens() — Check if more exist
  • nextToken() — Get next token
  • countTokens() — Count remaining
⚡ Faster than split() for simple cases
